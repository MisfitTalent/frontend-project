# Frontend Project

## Getting Started

Run the local development server:

```bash
npm run dev
```

Run the main validation commands before opening a pull request:

```bash
npm run lint
npm run build
```

## Repository Standards

Contribution and repository workflow standards are documented here:

- [Git workflow standard](/c:/Users/Kagiso/frontend-project/docs/git-workflow.md)
- [Contribution guide](/c:/Users/Kagiso/frontend-project/.github/CONTRIBUTING.md)

## Application Stack

- Next.js 16
- React 19
- TypeScript
- Ant Design

## Notes

Branch protection rules, required reviewers, and build validation still need to be configured in the remote GitHub and Azure DevOps repositories. Those settings cannot be enforced by repository files alone.
